# movie-app-server
Server application for the [movie-app](https://github.com/pembem22/movie-app) that fetches data from the IMDb GraphQL endpoint.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
